# AI Cover Letter Generator

An AI-powered SaaS utility that generates professional cover letters using Google Gemini AI.

## Features

- AI Cover Letter Generation
- Resume PDF Upload
- Resume Text Extraction
- Copy to Clipboard
- Recent History
- Clear History
- Loading State
- Responsive Design
- Environment Variable Security
- Retry Logic
- Fallback Letter Generation

## Tech Stack

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- Google Gemini API

## Setup

1. Clone repository
2. Install dependencies

npm install

3. Create .env

GEMINI_API_KEY=YOUR_KEY

4. Start server

node server.js
